Subnets Project

Subnets Project (HTML, CSS and JavaScript)

is a web tool designed to perform calculations related to computer networks, such as IP address conversion and subnet calculations. It is aimed at learning and practicing network concepts and front-end programming.

Main Features:

  • Interface developed with HTML and CSS.
  • Functionality implemented with JavaScript.
  • Conversion of IP addresses to binary.
  • Calculation of network addresses and planned future improvements (such as broadcast address).
